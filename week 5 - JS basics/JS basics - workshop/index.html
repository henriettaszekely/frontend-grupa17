JS workshop

Declare 5 object which will contain the following information about a product: name, price, quantity. The values for the products will be the following:

  1. flour, 5$, 1 piece;
  2. egg, 1$, 15 pieces;
  3. water, 2$, 4 pieces;
  4. chicken, 20$, 2 pieces;
  5. juice, 3$, 10 pieces;
  • Create a list named shoppingList which will contain all the products;
  • Calculate the total price for each product and add a new property, named total, which contains this value
  • Calculate the total price of the shopping list
  • Find the most expensive item of the shopping list, based on the total price of a product
  • Considering the the following items are in stock at the store: juice, water, egg, apple, milk, check which item of the shopping list is in stock and add an outOfStock property to each product which contains this information
